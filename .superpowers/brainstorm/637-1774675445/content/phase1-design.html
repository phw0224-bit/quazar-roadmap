동작 흐름

  1. / 입력 → 팝업 즉시 표시
  2. 텍스트 입력으로 블록 검색 필터링
  3. ↑↓ 탐색, Enter 선택
  4. Esc 또는 포커스 이탈 → 닫힘
  5. 블록 선택 시 /커맨드텍스트 자동 삭제
