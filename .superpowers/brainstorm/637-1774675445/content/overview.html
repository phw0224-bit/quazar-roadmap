Quazar Roadmap — 전체 기능 범위 파악

노션 수준 + 회사 맞춤 기능을 모두 구현하려면 아래처럼 큰 작업들이 남아 있습니다.
어디서 시작할지 먼저 정해야 효율적으로 진행할 수 있습니다.

✅ 이미 구현됨
칸반 보드 (4개 팀) 아이템 상세 패널 리치 텍스트 에디터 드래그&앤드롭 (DnD) People 보드 AI 요약 댓글 & 파일첨부 관계 아이템 링크 Supabase Realtime 다크모드 인증 (Auth)
🚧 구현 예정 기능 — 7개 Phase + 추가 기능
Phase 1 에디터 완성
  • 슬래시 커맨드 (/ 팝업)
  • Callout / Toggle 블록
  • 체크리스트 (TaskList)
  • 코드블록 하이라이팅
  • 블록 드래그 핸들 ⠿
Phase 2 필터 / 정렬 / 그룹화
  • 다중 필터 조건 (AND)
  • 정렬 (제목/날짜/상태)
  • 그룹화 (담당자/태그/상태)
  • URL 파라미터 저장
Phase 3 Cmd+K 전체 검색
  • 전문검색 (FTS, PostgreSQL)
  • 300ms 디바운스 실시간
  • 최근 방문 아이템
  • 클릭 → 상세 패널 이동
Phase 4 날짜 + 타임라인 뷰
  • 시작일/마감일 DatePicker
  • 우선순위 속성
  • 간트 차트 (CSS Grid + DnD)
  • 주/월/분기 줌
Phase 5 ⚠️ 대변경 사이드바 + 무한 중첩 페이지
  • 왼쪽 사이드바 트리 네비게이션
  • 무한 중첩 서브페이지
  • page/task/board 타입
  • 전체 레이아웃 재설계
Phase 6 @멘션 + Google Chat 알림
  • 댓글 @이름 자동완성
  • 멘션 시 Google Chat DM
  • 알림 히스토리
Phase 7 (장기) 커스텀 속성 시스템
  • 팀별 커스텀 필드 정의
  • 텍스트/숫자/날짜/선택 등
  • 수식(Formula) 필드
추가 (노션 수준) 미정의 추가 기능
  • Table / Gallery / List 뷰
  • Calendar 뷰
  • 템플릿 시스템
  • 북마크 / 즐겨찾기
  • 히스토리 & 버전 관리
  • 권한 관리 (팀별 접근)
